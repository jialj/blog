粘性页脚

sticky footer的三种实现方式：

sticky footer即让元素始终显示在页面底部，当页面内容不满页面高度时，指定元素在当前视口最下方；当页面内容超过页面高度时候，元素自动被挤到内容下方不发生覆盖。

方式一

<style>
  * {
      margin: 0px;
  }
  html,body{
      height: 100%;
  }
  .main-wrapper{
      min-height: 100%;
  }
  .main-wrapper:after{
      display: block;
      content: ' ';
      height: 0px;
      line-height: 0px;
      clear: both;
      visibility: hidden;
  }
  .mian{
      padding-bottom: 100px;
  }
  .footer{
      height: 100px;
      line-height: 100px;
      margin:-100px auto 0px auto;
      background-color: #008000;
  }
</style>
<body>
  <div class="main-wrapper">
      <div class="mian">I am content</div>
  </div>
  <div class="footer"> I am sticky footer </div>
</body>

原理：.main-wrapper元素的最小高度设置为100%，.mian元素的padding-bottom设置为100px,
.footer元素的margin-top设置为-100px。参考链接

方式二

<style>
  * {
      margin: 0;
  }
  html, body {
      height: 100%;
  }
  .wrapper {
      min-height: 100%;
      height: auto !important; /* 如果你不需要考虑IE6，则可以把这行与下一行代码删除 */
      height: 100%;
      margin: 0 auto -4em;
  }
  .footer, .push {
      height: 4em;
  }
  .footer, .push {
      clear: both;
  }
  .footer{
      background-color: #006DCC;
  }
</style>
<body>
  <div class="wrapper">
      <p>Your website content here.</p>
      <p>Your website content here.</p>
      <div class="push"></div>
  </div>
  <div class="footer">
      <p>Copyright (c) 2008</p>
  </div>
</body>

原理：和方式以类似，.wrapper最小高度为100%，且margin-bottom为-4em；.push和.footer高度都是4em。参考链接

方式三

<style type="text/css">
  *{
      margin: 0px;
  }
  html,body{
      height: 100%;
  }
  .page{
      min-height: 100%;
      display: flex;
      flex-direction: column;
  }
  .main{
      flex: 1;
  }
  .footer{
      height: 40px;
      line-height: 40px;
      background-color: #ccc;
  }
</style>
<body>
  <div class="page">
      <div class='main'>
          <p>I am contet</p>
          <p>I am contet</p>
          <p>I am contet</p>
      </div>
      <div class='footer'>
          I am footer
      </div>
  </div>
</body>

原理：利用flex布局，flex-derection设置为column表示纵向分布。.mian的flex设为1，.footer元素根据需要指定高度，.main即自适应占满page中除了footer之外的所有高度。参考链接
